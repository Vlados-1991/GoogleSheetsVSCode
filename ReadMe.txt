1. Открываем ссылку 'https://console.cloud.google.com' и соглашаемся с условиями.
2. Кликаем 'Выберите проект\Новый проект'
3. Вводим название проекта и нажимаем 'создать'
4. Выбираем созданный проекта
5. Переходим API и сервисы
6. Включить API и службы
7. Находим в поиске Google Sheets API и нажимаем включить
8. Открываем Credentials
9. Create Credentials
10. Service Account
11. Вводим имя и нажимаем создать
Сохраняем ключ в проект
